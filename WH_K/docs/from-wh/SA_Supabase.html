🔄 packages/sync-engine
• เก็บ pending ops ใน SQLite sync_queue
• ตรวจเน็ต → flush queue → ส่ง API
• Retry exponential backoff (3 ครั้ง)
• Dead-letter → แจ้ง user ด้วย banner
• Conflict resolution: last-write-wins
⚡ ประโยชน์ Monorepo
• แก้ bug ที่เดียว — ใช้ได้ทุก platform
• TypeScript type safety ครบทั้ง stack
• Deploy พร้อมกันด้วย Turborepo pipeline
• ทีมเดียวดูแล web + mobile + api

TRCloud Integration Layer
ข้อมูลที่แอปส่งไป TRCloud — format JSON ตาม API doc ของ TRCloud
POST — รับเข้าสินค้า (Goods Receipt)
// POST https://[company].trcloud.co/api/stock/gr
{
  "doc_type":     "GR",
  "po_ref":       "PO-2406-018",
  "warehouse_id": "DC-BKK",
  "received_by":  "user-uuid-xxx",
  "received_at":  "2026-06-19T13:42:00+07:00",
  "lines": [
    {
      "sku":     "FX-1180",
      "qty":     40,
      "bin":     "C-04-11",
      "unit":    "กล่อง",
      "lot":     "LOT-2026-06"
    }
  ],
  "offline_id":  "sync-uuid-yyy" // idempotency key
}
POST — จ่ายออก (Goods Issue)
// POST https://[company].trcloud.co/api/stock/gi
{
  "doc_type":    "GI",
  "so_ref":      "SO-2406-204",
  "dest":        "สาขารังสิต",
  "kind":        "TRANSFER",
  "issued_by":   "user-uuid-xxx",
  "issued_at":   "2026-06-19T09:30:00+07:00",
  "lines": [
    {
      "sku":    "TL-0241",
      "qty":    6,
      "bin":    "A-01-03",
      "unit":   "เครื่อง"
    }
  ],
  "offline_id": "sync-uuid-zzz"
}

Supabase + Google Cloud Services
Supabase รับผิดชอบ Data layer ทั้งหมด — Google Cloud Run รัน NestJS สำหรับ TRCloud integration
S
Supabase
แทน Cloud SQL + Firebase Auth + Redis + Cloud Storage Free 500MB · Pro $25/เดือน
Supabase Service แทนอะไรเดิม หน้าที่ในระบบ ราคา
Supabase Database
PostgreSQL 15 + RLS		 Cloud SQL ❌ ฐานข้อมูลหลัก · master data · transaction log · sync queue · Row Level Security ต่อ warehouse รวมใน plan
Supabase Auth
JWT + OAuth2		 Firebase Auth ❌ Login · Google SSO · JWT · RLS ผูกกับ user_id อัตโนมัติ · middleware Next.js + Expo รวมใน plan
Supabase Realtime
WebSocket subscriptions		 Redis Pub/Sub ❌ สต็อกอัปเดต live ทุกอุปกรณ์ทันที เมื่อ PC หรือมือถือ sync เสร็จ · subscribe ตาม warehouse_id รวมใน plan
Supabase Storage
S3-compatible		 Cloud Storage ❌ รูปสินค้า · ใบรับสินค้า PDF · export files · เข้าถึงผ่าน signed URL ฟรี 1GB
Supabase Edge Functions
Deno Deploy		 เพิ่มเติม ✨ Webhook รับ TRCloud callbacks · trigger แจ้งเตือนสต็อกต่ำ · pg_cron scheduled jobs ฟรี 500k req
G
Google Cloud
เหลือแค่ Cloud Run + ส่วนเสริม
บริการ หน้าที่ในระบบ ราคา/เดือน Priority
Cloud Run
NestJS + Docker		 รัน NestJS — TRCloud integration, sync service, report service · auto-scale 0→N ~$5–15 จำเป็น
Cloud Tasks
Async queue		 Queue offline ops จากมือถือ → retry อัตโนมัติ → ส่งต่อ TRCloud ~$1–3 แนะนำ
Secret Manager เก็บ TRCloud API key · Supabase service_role key · JWT secret ~$1 จำเป็น
Cloud Monitoring Logs + alerts สำหรับ Cloud Run errors, TRCloud sync failures ฟรี tier แนะนำ
