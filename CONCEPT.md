# คลังสำรับลาว — Kitchen Pantry System

> ระบบคลังโรงครัว โฟกัส **ต้นทุน real-time** ไม่ลงลึกเท่าระบบคลังพัสดุ/เหมือง
> Output หลักที่ Friday ใช้วัด: **ต้นทุนอาหารต่อหัวพนักงานต่อวัน (฿/คน/วัน)**

---

## ขอบเขต (Scope)

| ทำ | ไม่ทำ (เทียบคลังพัสดุ) |
|---|---|
| รับเข้า / เบิก → ตัดสต็อก real-time บนเว็บ | ผังคลัง, bin ละเอียด, cycle count เต็มรูปแบบ |
| แสดงต้นทุนทันทีเมื่อมีการเคลื่อนไหว | ABC ตามเครื่องจักร / WO แบบเหมือง |
| Sync สองทาง localStorage → GCP → TRCloud | Soft block ข้ามคลัง, handshake ซับซ้อน |
| โซ่ต้นทุน: เมนู → วัตถุดิบ → รอบเบิก → ฿/คน/วัน | รายงาน fraud / variance ขั้น audit |

---

## การรับข้อมูล (Input)

```
รับเข้า (GR)          เบิก (GI)
─────────────────     ─────────────────
📷 ถ่ายรูปเป็นหลัก      🔍 สแกนบาร์โค้ดเป็นหลัก
   (ใบส่งของ, ของจริง)     (วัตถุดิบตอนตัดออก)
✏️ พิมพ์ PO / รายการ      เลือกเมนู / รอบเบิก (optional)
```

ทั้งสองทาง → ข้อมูล digital เหมือนกัน → ตัด stock + คำนวณต้นทุนบนเว็บทันที

---

## Data Flow

```
┌─────────────┐    change-on-edit    ┌──────────────┐    sync สองทาง    ┌──────────┐
│ localStorage │ ──────────────────► │ Google Cloud │ ◄──────────────► │ TRCloud  │
│ (offline)    │    เติมส่วนที่หาย     │ (ฐานกลาง)     │   เช็คไม่ตรง→เติม  │ (บริษัท)  │
└─────────────┘                      └──────────────┘                   └──────────┘
```

- บันทึกลงเครื่องก่อน (ผ่านก่อน บันทึกไว้) แล้วค่อย push
- Sync แบบ **change-on-edit**: แก้ฝั่งไหน ฝั่งนั้นเป็นต้นทาง อีกฝั่งตามเติมส่วนที่หาย

---

## โซ่ต้นทุน (Cost Chain)

```
เมนู (Menu)
  └── วัตถุดิบ + ปริมาณต่อหน่วย (BOM)
        └── รอบการเบิก (Withdrawal Round / มื้ออาหาร)
              └── ต้นทุนรวมรอบนั้น (Cost Snapshot ณ วันเบิก)
                    └── ÷ จำนวนพนักงานที่ทำงานวันนั้น
                          └── ★ ต้นทุน/คน/วัน (฿/คน/วัน) ← Output จริงของโรงครัว
```

**Cost Snapshot** — ราคาทุนล็อก ณ วันเบิก ไม่เปลี่ยนหลังยืนยัน (ใช้คำนวณย้อนหลังได้)

---

## หัวใจร่วมกับคลังพัสดุ

- ตัด stock แบบ **real-time** บนเว็บ/แอป
- โครง sync engine เดียวกัน (localStorage → GCP → TRCloud)
- Master data สินค้า/หน่วย จาก TRCloud
